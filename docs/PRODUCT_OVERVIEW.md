# 競馬チャンプ - プロダクト概要

## 🎯 このサービスは何？

**競馬チャンプ**は、ブラウザで遊べるリアルな競馬予想ゲームです。

実際の競馬レースを使って、ゲーム内コインで予想を楽しめます。お金は一切かかりません。コインは無料で獲得でき、的中すればコインが増える仕組みです。

**アカウント登録（無料）が必要です。** メールアドレスまたはGoogle/Twitterアカウントで簡単に登録できます。

## ✨ 主な特徴

### 1. 🎮 無料で楽しめる

- **初期コイン**: 新規登録で10,000コインプレゼント
- **コイン獲得方法**:
  - 予想が的中するとコインが増える
  - 毎日ログインでボーナスコイン
  - 広告を見てコイン獲得
- **お金は一切かかりません**: コインは換金できませんが、無料で楽しめます

### 2. 🏇 実際の競馬データを使用

- **リアルなレース**: 実際に開催されているJRAのレースを使用
- **リアルタイムオッズ**: 実際のオッズをリアルタイムで取得
- **本物の出走馬**: 実際の馬、騎手、調教師の情報
- **リアルな結果**: レース終了後、実際の結果で判定

### 3. 📊 6種類の予想タイプ

実際の競馬と同じように、オッズが取得できるすべての予想方法に対応しています：

- **単勝**: 1着を予想（シンプルでわかりやすい）
- **複勝**: 3着以内を予想（的中しやすい）
- **馬連**: 1着・2着の組み合わせを予想（配当が大きい）
- **ワイド**: 3着以内の2頭の組み合わせを予想（バランス型）
- **3連複**: 3着以内の3頭の組み合わせを予想（順番は問わない）
- **3連単**: 1着・2着・3着の順番を正確に予想（最高配当）

### 4. 🏆 全国ランキング

- **総資産ランキング**: コインをたくさん持っている人が上位
- **収支ランキング**: 儲けが多い人が上位
- **的中率ランキング**: 予想が当たりやすい人が上位
- **週間・月間ランキング**: 期間限定のランキングもあり
- **報酬**: 上位に入るとコインやバッジがもらえる

### 5. 🎁 ボーナス・報酬システム

- **デイリーボーナス**: 毎日100コイン
- **ログインボーナス**: 連続ログインで追加コイン
- **ランキング報酬**: 上位入賞でコインやバッジ
- **バッジシステム**: 実績をバッジで表示

## 🎮 使い方（簡単3ステップ）

### ステップ1: アカウント作成（無料・必須）

1. メールアドレスで登録、またはGoogle/Twitterでログイン
2. 初回ログインで**10,000コイン**プレゼント
3. すぐにプレイ開始可能

**アカウント登録は必須です。** データが保存され、ランキングに参加できます。

### ステップ2: レースを選んで予想

1. **レース一覧**から予想したいレースを選択
2. **予想タイプ**を選ぶ（単勝、複勝、馬連、ワイド、3連複、3連単）
3. **馬を選ぶ**
   - 単勝・複勝: 1頭
   - 馬連・ワイド: 2頭
   - 3連複: 3頭（順番は問わない）
   - 3連単: 3頭（1着・2着・3着の順番を指定）
4. **賭け金を決める**（10コイン〜10,000コイン）
5. **予想確定**

### ステップ3: 結果を待ってコイン獲得

1. レース開始まで待機
2. レース終了後、自動的に結果判定
3. **的中したら**: 賭け金 × オッズ = 獲得コイン
4. **不的中なら**: 賭け金は没収（コインは減ります）

## 💰 コインシステム

### コインの使い道

- **予想購入**: レースの予想をするために使用
- **プレミアム機能**: 詳細な分析データを見る（オプション）

### コインの獲得方法

| 方法 | 獲得コイン | 条件 |
|------|----------|------|
| 予想的中 | 賭け金 × オッズ | 予想が当たった時 |
| デイリーボーナス | 100コイン | 1日1回、ログイン時 |
| ログインボーナス | 50〜2,000コイン | 連続ログイン日数に応じて |
| 広告視聴 | 50コイン | 1日最大5回 |
| ランキング報酬 | 5,000〜50,000コイン | ランキング上位入賞 |

### コインの購入（オプション）

- コインは無料で獲得できますが、追加で購入することも可能
- 購入したコインでより多くの予想ができます
- **換金はできません**（ゲーム内通貨のみ）

---

## 🏆 ランキングシステム

### ランキングの種類

1. **総資産ランキング**
   - 現在持っているコインの数で順位が決まります
   - コインをたくさん持っている人が上位

2. **収支ランキング**
   - 獲得したコイン - 使ったコイン = 収支
   - 儲けが多い人が上位

3. **的中率ランキング**
   - 的中数 ÷ 予想数 × 100 = 的中率
   - 予想が当たりやすい人が上位

4. **週間・月間ランキング**
   - 期間限定のランキング
   - 毎週・毎月リセットされて新しい競争が始まります

### ランキング報酬

| 順位 | 報酬 |
|------|------|
| 🥇 1位 | 50,000コイン + ゴールドバッジ |
| 🥈 2-3位 | 30,000コイン + シルバーバッジ |
| 🥉 4-10位 | 10,000コイン + ブロンズバッジ |
| 11-100位 | 5,000コイン |

## 🎁 バッジシステム

実績をバッジで表示できます：

- **初勝利**: 初めて予想が的中
- **10連勝**: 10回連続で的中
- **大逆転**: 100倍以上のオッズで的中
- **コレクター**: 全競馬場で予想
- **常連**: 30日連続ログイン
- **ランキングバッジ**: ランキング上位入賞

バッジはプロフィールページに表示され、他のユーザーにも見えます。

## 💎 プレミアム機能（オプション）

月額980円で、以下の特典が受けられます：

- ✅ **広告非表示**: 快適にプレイ
- ✅ **詳細分析データ**: 過去成績、血統情報など
- ✅ **予想上限引き上げ**: 通常10,000コイン → 50,000コイン
- ✅ **コイン購入20%割引**: コインを安く購入
- ✅ **限定バッジ・アバター**: プレミアム会員限定
- ✅ **優先サポート**: 問題解決が早い

**プレミアムなしでも完全に楽しめます**。追加機能として提供しています。

## ❓ よくある質問

### Q1. お金はかかりますか？

**A.** 基本は完全無料です。アカウント作成で10,000コインもらえますし、予想が的中すればコインが増えます。コインの追加購入やプレミアム機能はオプションです。

### Q2. コインは換金できますか？

**A.** いいえ、コインは換金できません。ゲーム内通貨のみです。これは法律上の制約のためです。

### Q3. 実際の競馬で使えますか？

**A.** いいえ、これはゲームです。実際の競馬の購入には使用できません。予想の練習や楽しみのためのサービスです。

### Q4. どのくらいの頻度でレースがありますか？

**A.** 実際のJRAのレーススケジュールに合わせます。通常、週末を中心に開催されます。

### Q5. 予想はいつまでできますか？

**A.** レース開始の5分前まで予想できます。それ以降は予想受付を終了します。

### Q6. コインがなくなったらどうなりますか？

**A.** 以下の方法でコインを獲得できます：
- デイリーボーナス（毎日100コイン）
- ログインボーナス
- 広告視聴（50コイン/回）
- ランキング報酬
- コイン購入（オプション）

### Q7. アカウント作成は必須ですか？

**A.** はい、アカウント作成は必須です。以下の理由があります：
- データが保存される（予想履歴、コイン残高など）
- ランキングに参加できる
- コイン購入が可能
- バッジや実績が記録される

メールアドレスまたはGoogle/Twitterアカウントで簡単に登録できます。

### Q8. スマートフォンで使えますか？

**A.** はい、ブラウザでアクセスできるので、スマートフォンでも利用できます。将来的にはアプリ版も検討しています。

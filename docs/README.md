# 競馬チャンプ - プロジェクト概要

## プロジェクト概要

**競馬チャンプ**は、ブラウザで遊べる競馬予想ゲームです。実際の国内競馬レースデータを使用し、リアルタイムでオッズを取得してゲーム内コインで予想を楽しめます。アカウント登録（無料）が必要です。

## 主な機能

- 🎮 **ゲーム内コインシステム**: コインを使って競馬予想が可能
- 🏇 **リアルタイムレースデータ**: 実際の国内競馬レースデータを使用
- 📊 **リアルタイムオッズ**: 実際のオッズをリアルタイムで取得・反映
- 🏆 **全国収支ランキング**: ユーザー同士で競えるランキング機能

## 技術スタック

### フロントエンド
- Next.js (最新版, App Router)
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zustand (状態管理)
- React Query (データフェッチング)

### バックエンド
- FastAPI (Python) - バックエンドAPI（スクレイピング・データ取得・予想処理等）
- Supabase Auth - 認証システム
- Supabase Python SDK - データベースアクセス
- PostgreSQL (メインデータベース) - Supabase
- Redis (キャッシュ・セッション管理)
- Celery (ジョブキュー) - Python用

### データ取得
- Playwright (Python) - スクレイピング
- BeautifulSoup4 - HTMLパーシング
- APScheduler - 定期実行

### インフラ
- Vercel (フロントエンドホスティング)
- Railway / Render (FastAPIホスティング)
- Upstash (Redis)
- Supabase (PostgreSQL + Auth)

## 開発フェーズ概要

1. **Phase 1: 基盤構築** (2週間)
2. **Phase 2: データ取得システム** (2週間)
3. **Phase 3: ゲーム機能実装** (3週間)
4. **Phase 4: ランキング・ソーシャル機能** (1週間)
5. **Phase 5: マネタイズ機能** (2週間)
6. **Phase 6: 最適化・デプロイ** (1週間)

**合計: 約11週間（3ヶ月弱）**

## マネタイズ概要

複合型の収益モデルを採用：

- **広告収入**: Google AdSense統合
- **ゲーム内コイン購入**: 課金システム
- **プレミアムサブスクリプション**: 月額制
- **アフィリエイト・パートナーシップ**: 競馬関連サービス
